### Architecture

- **`find_node_mut`** — helper to locate node, returns TASK_NOT_FOUND if absent
- **`preflight_mutation`** — shared: `load_validate_reconcile()` + desync check
- **`apply_simple_transition`** — shared: complete/fail/block (status validation, reason field set, event emit)
- **`persist_mutation`** — shared: call reconcile, bump revision, atomic persist (events before graph)

### Error Handling

- `TASK_NOT_FOUND` — node does not exist
- `INVALID_TRANSITION` — command called on unsuitable status
- `LEASE_NOT_OWNED` — non-owner accessing IN_PROGRESS node
- `STALE_REVISION` — client's revision ≠ current graph revision
- `EVENT_LOG_DESYNC` — desync detected during preflight
